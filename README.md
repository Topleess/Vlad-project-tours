# Веб-приложение для туристического агентства

Курсовой проект по теме "Разработка функциональной части веб-приложения для туристического агентства".

## Технологический стек

### Бэкенд (Backend)
- Язык программирования: Python
- Фреймворк: FastAPI
- СУБД: SQLite
- ORM: SQLAlchemy
- Миграции: Alembic
- Аутентификация: JWT токены

### Фронтенд (Frontend)
- Стандарты: HTML5, CSS, JavaScript
- Фреймворк: Vue 3
- Метарамка (SSR/роутинг): Nuxt 3
- CSS-фреймворк: Tailwind CSS
- Библиотека визуализации данных: D3.js

## Функциональность

1. **Регистрация и авторизация пользователей**
   - Регистрация нового клиента
   - Авторизация (вход на сайт)
   - Восстановление пароля

2. **Поиск и каталог туров**
   - Просмотр каталога туров
   - Фильтрация и поиск туров
   - Детальная информация о турах

3. **Бронирование туров**
   - Оформление бронирования
   - Выбор дополнительных услуг
   - Оплата тура (имитация)
   - Управление бронированиями

4. **Личный кабинет**
   - Профиль пользователя
   - История заказов
   - Отзывы о поездках

5. **Админ-панель**
   - Управление списком туров
   - Управление бронированиями
   - Управление пользователями

## Функциональность системы аутентификации и администрирования

### Авторизация и регистрация
- Регистрация новых пользователей с валидацией полей
- Авторизация с использованием JWT-токенов
- Восстановление пароля
- Уровни доступа (роли): клиент, администратор

### Личный кабинет пользователя
- Просмотр и редактирование личных данных
- Изменение пароля
- Просмотр истории бронирований
- Управление текущими бронированиями

### Административная панель
- Дашборд с ключевыми показателями и статистикой
- Управление пользователями:
  - Просмотр списка всех пользователей с пагинацией и поиском
  - Редактирование данных пользователей
  - Назначение ролей и прав доступа
  - Блокировка/разблокировка пользователей
- Управление турами:
  - Добавление новых туров
  - Редактирование существующих туров
  - Удаление туров
  - Категоризация и фильтрация туров
- Управление бронированиями:
  - Просмотр всех бронирований
  - Подтверждение/отмена бронирований
  - Просмотр статистики по бронированиям

## Запуск проекта

### Бэкенд

1. Установите зависимости:
```bash
cd backend
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
cd backend
docker-compose up -d
```

Бэкенд будет доступен по адресу: http://localhost:8000/api
API-документация доступна по адресу: http://localhost:8000/docs

### Фронтенд

1. Установите зависимости:
```bash
cd frontend
npm install
```

2. Запустите приложение:
```bash
cd frontend
npm run dev
```

Фронтенд будет доступен по адресу: http://localhost:3000

## Тестовые учетные записи

- Администратор: admin@example.com / admin123
- Пользователь: user@example.com / user123

## Доступ к административной панели

После регистрации пользователя, вам нужно назначить ему роль администратора через API или базу данных:

```sql
UPDATE users SET is_admin = true WHERE email = 'your_email@example.com';
```
